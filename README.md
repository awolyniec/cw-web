# ChattyWatty Web App

### Description
A single-room text chat app. Pick a name, a color, and chat with other people in the room.

### How to run

Other services needed:
- cw-server 

How to run cw-web:

You should have Node 12, npm 6, and yarn 1 installed (other versions not tested).

Run these commands:
```yarn install```
```yarn start```
